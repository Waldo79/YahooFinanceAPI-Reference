# Capture Guidelines

## Recommended Capture Set

| Type | Symbols |
|---|---|
| Stocks | AAPL, MSFT |
| ETFs | SPY, QQQI |
| Closed-end funds | NAC, PDI |
| Mutual funds | XNACX, VTSAX |
| Futures | ZT=F, CL=F |
| Indices | ^GSPC, ^DJI |
| Crypto | BTC-USD, ETH-USD |
| Forex | EURUSD=X, JPY=X |
