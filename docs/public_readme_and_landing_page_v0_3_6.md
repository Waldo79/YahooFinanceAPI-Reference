# v0.3.6 Public README and GitHub Landing Page

## Purpose

This release makes the repository easier for first-time public users to understand. It explains what the project tracks, what it does not do, how to use the data files, and how to submit observations.

## Main framing

YahooFinanceAPI-Reference is a public reference and change-tracking project for observed Yahoo Finance API behavior.

Scripts and validators are support tools. The main product is the public record of observed endpoint behavior, field behavior, symbol behavior, result states, and change records.

## Key additions

- Public-facing README.md
- First-time user guide
- Observation reporting guide
- GitHub README checklist
- Landing page section table

## Master database impact

No new Yahoo API field rows were added in this release. The master field database remains at 198 rows.
